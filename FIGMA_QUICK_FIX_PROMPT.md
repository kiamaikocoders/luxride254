# Quick Fix Prompt for Figma - Make Design Smooth

Copy and paste this entire prompt into Figma AI to quickly improve your design:

---

**PROMPT TO USE IN FIGMA:**

```
Refine the entire LuxeRide VIP app design to make it smoother and more premium:

COLOR FIXES:
- Replace all gold/yellow colors with exact #FFD700
- Ensure background is pure black #0a0a0a (not dark gray)
- Make card backgrounds #1a1a1a consistently
- Use #71717a for all secondary text
- Ensure borders are #27272a

SPACING IMPROVEMENTS:
- Increase spacing between all elements (minimum 16px between cards, 24px between sections)
- Add consistent padding: 16px inside cards, 24px on screen edges
- Ensure all elements align to 8px grid
- Add more breathing room in forms (24px between fields)

TYPOGRAPHY:
- Make headings bolder and larger (24px for h2, 18px for h3)
- Increase line height to 1.5x for body text
- Use consistent font sizes throughout
- Make button text bold and uppercase with letter spacing

COMPONENT POLISH:
- Increase border radius to 12px for buttons, 16px for cards
- Add subtle shadows to all cards: 0 2px 8px rgba(0,0,0,0.3)
- Make all buttons at least 48px height for better touch targets
- Add gold border (2px) on input focus states
- Ensure all interactive elements have hover/press states

BOTTOM NAVIGATION:
- Make tab bar 64px height
- Increase icon sizes (28px active, 24px inactive)
- Add gold underline or background for active tab
- Add subtle shadow above tab bar

BUTTONS:
- Make primary buttons full-width gold (#FFD700) with white bold text
- Increase button height to 56px
- Add subtle shadow: 0 4px 12px rgba(255,215,0,0.2)
- Add smooth press animation (scale to 0.98x)

CARDS:
- Add consistent 16px padding inside all cards
- Use 16px border radius
- Add subtle shadow for depth
- Increase spacing between cards to 16px minimum

MAP SCREENS:
- Make maps full-screen with dark theme
- Add semi-transparent dark overlay on non-interactive areas
- Make bottom sheets slide up smoothly with rounded top corners (24px)
- Ensure controls overlay elegantly on map

ANIMATIONS:
- Add smooth transitions: 200ms ease-in-out for all interactions
- Add subtle scale animation on button press (0.98x)
- Add smooth slide animations for screen transitions
- Add fade-in for content loading

VISUAL HIERARCHY:
- Make important elements larger and more prominent
- Use gold (#FFD700) only for primary actions and highlights
- Ensure proper contrast for readability
- Add visual separation between sections

ICONS:
- Make all icons consistent size (24px tab bar, 20px inline)
- Use gold for active states, gray for inactive
- Add proper spacing around icons (8px minimum)

Remove any cluttered elements, improve alignment, and ensure the entire design feels premium, luxurious, and smooth. Focus on consistency, proper spacing, and maintaining the dark theme with gold accents throughout.
```

---

## How to Use:

1. **Open your Figma file**
2. **Select all frames/screens** (or specific ones you want to refine)
3. **Use Figma's AI features** (First Draft, or the AI assistant)
4. **Paste the prompt above**
5. **Apply the changes**

## Alternative: Apply to Specific Screens

If you want to refine one screen at a time, use this shorter version:

```
Refine this screen: Increase spacing (16px minimum between elements), use exact gold color #FFD700, add subtle shadows to cards, make buttons 48px height, improve typography hierarchy, add smooth animations, and ensure premium luxury feel with consistent dark theme.
```

---

## Most Common "Not Smooth" Issues:

1. **Inconsistent spacing** → Use 8px grid, minimum 16px between elements
2. **Wrong colors** → Use exact #FFD700 for gold, #0a0a0a for background
3. **Small touch targets** → Make buttons at least 48px height
4. **No animations** → Add 200ms transitions to all interactions
5. **Poor hierarchy** → Make important elements larger and more prominent
6. **Inconsistent styling** → Use same border radius, shadows, padding throughout

Apply these fixes and your design will feel much smoother!
